# smart-finance-analytics-platform
Full-stack finance management platform using Spring Boot, React, MySQL, and Docker with expense tracking, budgeting, analytics dashboards, and JWT-based authentication.
## Features
User Authentication & Authorization (JWT)
Expense Tracking
Budget Management
Analytics Dashboard
Role-Based Access Control
RESTful APIs
Dockerized Deployment
Responsive UI
